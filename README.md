# Vitalink – Real-Time INR Monitoring System

Vitalink is a smart healthcare monitoring application designed to track patient INR levels in real time and securely communicate the data to hospitals for faster medical response.

## 🚀 Features

- Real-time INR monitoring
- Instant hospital alerts for abnormal readings
- Secure patient data handling
- Live dashboard for patient status
- Fast and responsive user interface

## 🛠️ Tech Stack

### Frontend
- React.js
- TypeScript
- Tailwind CSS

### Backend
- Node.js
- Express.js

## 📌 Purpose

The main goal of Vitalink is to improve patient safety by enabling continuous monitoring and quick communication between patients and healthcare providers.
